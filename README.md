# Sittich
Ein Sittich ist ein meist kleiner bis mittelgroßer, farbenfroher Papagei mit langem Schwanz, der als geselliger Schwarmvogel in Savannen und Wäldern Australiens, Südamerikas und Asiens lebt
. Bekannte Arten wie der Wellensittich sind intelligent, soziale Vögel, die in Gefangenschaft viel Freiflug benötigen und selten allein gehalten werden sollten
